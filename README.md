# Currency Converter API

Простое приложение для конвертации валют на FastAPI с JWT аутентификацией и интеграцией с внешним API обменных курсов.

## Особенности проекта

- ✅ JWT аутентификация (регистрация/вход)
- ✅ Интеграция с внешним API обменных курсов
- ✅ Конвертация валют в реальном времени
- ✅ Список поддерживаемых валют
- ✅ SQLite база данных через SQLAlchemy ORM
- ✅ Pydantic модели для валидации данных
- ✅ Обработка ошибок
- ✅ Модульные тесты
- ✅ Переменные окружения для конфигурации
